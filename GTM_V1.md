# StillMind GTM v1

## 决策

GTM v1 不推广“完整沉寂小我体系”，只验证一个楔子：

> 情绪触发后，StillMind 能否让用户在 60 秒内从角色位退回观察位，并推迟一次冲动回应？

## 市场位置

公开市场上的邻近品类已经教育了用户：

| 邻近品类 | 用户已理解的价值 | StillMind 不做 | StillMind 借用的习惯 |
| --- | --- | --- | --- |
| AI 情绪陪伴 | 输入困扰并获得回应 | 长对话、诊断、安慰人格化 | 低门槛输入 |
| 冥想/呼吸 | 跟随节奏降低激活 | 内容库和长练习 | 呼吸仪式感 |
| 情绪日记 | 记录和回顾情绪 | 事后日志为核心 | 本地历史与复盘 |
| ACT/认知解离练习 | 与念头拉开距离 | 学术课程式表达 | 第三人称与观察者机制 |

StillMind 的差异不是发明新的心理机制，而是把它压缩成一条可记忆的产品语言：

`内在电影 → 观众席 → 念头字幕 → 回到现实`

当前市场结论为定性判断。实时市场搜索接口在本轮被 403 阻断，因此本文件不写未经新鲜来源验证的市场规模数字。

## 首批用户

优先人群：

- 22-40 岁知识工作者、创作者和自我觉察人群；
- 经常在批评、冲突、被忽视或“必须证明自己”时立刻回复；
- 愿意尝试正念/自我观察，但不想进入长冥想或心理聊天。

不作为首批目标：

- 正在寻求诊断、治疗或危机支持的人；
- 只寻找泛冥想内容库的人；
- 一开始就要求完整灵修体系的人。

## 使用时刻

只抢一个时刻：

> 冲突或批评刚发生，用户准备立即回复之前。

主承诺：

> 把脑子里的剧情，变成一场可以退出的电影。

解释句：

> 不是让你想开点，而是让你看见：你正在入戏。

## 首轮渠道

第一轮只做 founder-led concierge GTM：

1. 3-4 位会诚实反馈的朋友；
2. 2-3 位产品/设计同行；
3. 1-2 位心理、冥想或觉察背景用户；
4. 1-2 位弱关系用户，用来检验邀约文案。

使用 [INVITATIONS.md](./INVITATIONS.md) 的话术逐个邀请，不群发广告，不投放。

## 可验证指标

首轮样本目标：10 人。这个样本只用于发现方向，不用于证明统计显著性。

| 漏斗 | 定义 | 首轮通过线 |
| --- | --- | --- |
| 接受测试 | 明确愿意打开链接 | ≥ 7/10 |
| 完成体验 | 到达 Return 页 | ≥ 6/10 |
| 核心价值 | “多了一点距离”回答“有/一点” | ≥ 4/10 |
| 真实场景 | 使用真实触发而非一键演示 | ≥ 3/10 |
| 复用信号 | 7 天内主动再用一次 | ≥ 2/10 |
| 强信号 | 主动转发、追问或愿意付费之一 | ≥ 2/10 |

North Star 的早期代理指标不是停留时长，而是：

> 完成 Observer Mode 后，回答“有/一点距离”的真实触发会话数。

## 记录方式

首轮不建账号和远程用户数据库：

- 产品本地记录会话；
- Return 页让用户选择距离感并复制匿名反馈；
- 创始人把反馈手工记录在一个表格：日期、来源、是否真实触发、是否完成、距离感、是否复用、一句话反馈；
- 不记录用户的原始 trigger 文本到外部表格。

## 决策规则

继续强化 Inner Cinema：

- ≥ 4/10 出现核心价值信号；且
- ≥ 2/10 在 7 天内复用或主动要求继续。

先修体验而不是扩方法库：

- 用户理解价值但完成率 < 60%；或
- 多数人说文字多、节奏慢、像 demo。

重新审视定位：

- < 3/10 完成；或
- < 2/10 感到任何距离；或
- 反馈主要是“好看/有创意”，没有真实使用时刻。

## 本轮明确不做

- 登录、云同步、支付；
- 完整小我降噪方法库；
- 临床心理或治疗承诺；
- 大规模投放；
- 为了“像完整 App”而增加无验证价值的页面。

## 从 GTM v1 到完整产品

只有 Inner Cinema 通过核心价值和复用验证后，才按真实需求增加第二种练习。完整产品的方向仍是“沉寂小我的多种方法集合体”，但每个新方法都必须回答：它服务哪个触发时刻，是否比 Inner Cinema 更有效，以及用户是否会重复使用。
