# StillMind Support

StillMind is a general-wellness, self-reflection product. It does not provide diagnosis, therapy, medical advice, crisis response, or emergency support.

## Product Feedback

Use GitHub issues for non-sensitive product reports:

- bugs, broken flows, confusing copy, accessibility problems;
- device, browser, or App version context;
- the screen or step where the problem happened.

Do not include private trigger text, contact information, medical history, crisis details, or anything you would not want to be public. GitHub issues are public by default.

## Privacy and Local Data

The current product is local-first. You can export or delete local practice data from the app's Profile screen. Web demo data can be removed by clearing this site's browser storage.

Private privacy-request handling requires a monitored support mailbox before broad App Store launch. Until that mailbox is active, do not send personal emotional content through GitHub.

## Immediate Danger

If you or someone else may be in immediate danger, do not wait for StillMind or a GitHub response. Contact local emergency services, go to an emergency department, or ask a trusted person to stay with you.
